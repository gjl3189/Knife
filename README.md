# Knife

* [x] 在内部储存和外部储存卡之间移动文件
* [x] 基于目录播放的视频播放器
* [ ] 基于目录播放的音乐播放器
* [x] 基于目录浏览的图片浏览器
* [x] Web 服务器，可用于在设备之间传输文件和播放视频
* [ ] 深度清理储存空间
* [x] 压缩和解压文件
* [x] 格式化 Epub 文件名

[下载](https://github.com/grandiloquent/Knife/blob/master/app/release/app-release.apk?raw=true)
